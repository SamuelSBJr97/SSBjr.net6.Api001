SSBJr.Net6.Api001 (API)
=======================

Pasta do projeto da API.

Como rodar (desenvolvimento)

```powershell
cd src\src\SSBJr.Net6.Api001
dotnet restore
dotnet run
```

Endpoints principais
- GET /api/products
- POST /api/products

Swagger estará disponível em /swagger quando a aplicação estiver rodando.
